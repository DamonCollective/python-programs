# Alegro.gr — Product Description Standards

## Voice & Style Rules
- Greek text uses "φτιάχνεται" (not "Χειροποίητη") for "made in Greece"
- Mention "Σε πολλά χρώματα, εδώ σε {color_neuter}" — always acknowledge the range
- Closing line: "φτιαγμένη για να αντέχει χρόνια" (not "χωρίς να χάνει το σχήμα της")
- Use cases (EL): cosplay, Halloween, απόκριες, retro / vintage look, μεταμφιέσεις, role-playing
- No "45 χρόνια παράδοσης" — just "Φτιάχνεται στο εργαστήριό μας στην Αθήνα"
- Bullet list at end: 4 items (Στυλ, Κατάλληλη για, Ελαστική βάση, Κατασκευάζεται)
- Related products field: all wigs of the same product line (Cecilia → all Cecilias, etc.)
